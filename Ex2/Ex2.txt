1- A query string contains data, and can be put into the URI. It is used to send or request data on the server side of a website, such as in forms.
2- It is characteristic of both methods, but in the GET method, the query string is found in the URL, and in the POST method, it is in the HTTP message body. In the example above, the query string is characteristic of the GET method.
3-"age=22&name=Obi+Wan+Kenobi&number=5435432156" is the part of the URL above that is the query string.
